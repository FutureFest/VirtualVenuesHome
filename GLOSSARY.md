# Glossary

Project terminology and domain-specific vocabulary used across VirtualVenues projects.

## Core Concepts

| Term | Definition |
|------|------------|
| Venue | A virtual space where events or gatherings take place |
| Event | A scheduled occurrence within a venue |
| Attendee | A user participating in an event |
| Host | A user who creates and manages events |
| Session | A distinct segment or activity within an event |
